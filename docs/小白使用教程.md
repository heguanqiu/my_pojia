# Codex Session Patcher 一键安装使用指南

这份文档面向第一次使用的用户，按步骤操作即可完成安装和启动。

## 适用系统

- macOS
- Linux
- Windows WSL

## 准备环境

必须安装：

- Python 3.8 或更高版本

如果要使用 Web UI，还需要安装：

- Node.js
- npm

## 一键安装

打开终端，执行：

```bash
git clone https://github.com/ryfineZ/codex-session-patcher.git
cd codex-session-patcher
./scripts/install.sh
```

脚本会让你选择安装内容：

```text
1) Web UI + CLI（推荐，图形界面，适合新手）
2) 只安装 CLI（命令行，依赖更少）
```

新手推荐选择 `1`。

## 安装完成后怎么用

### 启动 Web UI

如果你安装了 Web UI，执行：

```bash
codex-patcher-web
```

然后打开浏览器访问：

```text
http://127.0.0.1:8080
```

### 使用 CLI

查看帮助：

```bash
codex-patcher --help
```

清理最新会话：

```bash
codex-patcher --latest
```

预览清理效果，不修改文件：

```bash
codex-patcher --latest --dry-run --show-content
```

查看 CTF 配置状态：

```bash
codex-patcher --ctf-status
```

安装 Codex CTF Profile：

```bash
codex-patcher --install-ctf-config
```

安装后用 CTF Profile 启动 Codex：

```bash
codex -p ctf
```

## 非交互安装

如果你想直接安装 Web UI：

```bash
./scripts/install.sh --web --yes
```

如果你只想安装 CLI：

```bash
./scripts/install.sh --cli --yes
```

安装完成后立即启动 Web UI：

```bash
./scripts/install.sh --web --yes --start
```

## 常见问题

### 提示 codex-patcher: command not found

先执行：

```bash
export PATH="$HOME/.local/bin:$PATH"
```

然后再试：

```bash
codex-patcher --help
```

如果仍然不行，重新打开一个终端。

### 提示缺少 python3-venv

Ubuntu 或 Debian 用户可以执行：

```bash
sudo apt install python3-venv
```

如果你使用的是特定 Python 版本，例如 Python 3.12，可以执行：

```bash
sudo apt install python3.12-venv
```

然后重新运行安装脚本：

```bash
./scripts/install.sh
```

### 提示缺少 Node.js 或 npm

如果只想使用命令行版本，可以在安装时选择 `只安装 CLI`。

如果要使用 Web UI，请先安装 Node.js：

```bash
# macOS
brew install node

# Ubuntu / Debian
sudo apt install nodejs npm
```

安装完成后重新运行：

```bash
./scripts/install.sh
```

### Web UI 端口被占用

默认地址是：

```text
http://127.0.0.1:8080
```

如果 8080 端口被占用，可以直接用 uvicorn 指定其他端口：

```bash
cd codex-session-patcher
.venv/bin/python -m uvicorn web.backend.main:app --host 127.0.0.1 --port 8081
```

然后访问：

```text
http://127.0.0.1:8081
```

## 卸载

删除启动命令：

```bash
rm -f ~/.local/bin/codex-patcher ~/.local/bin/codex-patcher-web
```

删除项目目录即可卸载程序：

```bash
rm -rf codex-session-patcher
```

如果你不再需要配置文件，也可以删除：

```bash
rm -rf ~/.codex-patcher
```
